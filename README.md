## ðŸš€ Project Overview

This dashboard helps answer key business questions:

- Which **branches** generate the highest sales?
- What are the top-performing **product lines**?
- Are **members** spending more than **normal customers**?
- Which **city** has the strongest performance?
- What are the **peak sales hours**?
- Which **payment method** is used the most?
- How profitable is each **product category**?

The goal of this project is to visualize sales data in a clean, efficient, and interactive format suitable for decision-makers.
